# Taller de Python y NumPy

Este repositorio contiene ejercicios prácticos de Python y NumPy.

---

## 📌 Contenido

### 1. Python básico
- Listas
- Tuplas
- Diccionarios
- Funciones
- Comprehensions
- map y filter

---

### 2. NumPy
- Creación de arrays
- Operaciones con vectores y matrices
- Arrays aleatorios
- Atributos (.shape, .dtype, etc.)

---

### 3. Manipulación de datos
- Indexing y slicing
- Boolean masking
- Reshape
- Concatenación

---

### 4. Operaciones numéricas
- Suma, resta, multiplicación
- Broadcasting
- Funciones matemáticas (sqrt, abs, clip)
- Reducciones (sum, mean, max, min)

---

## 🎯 Objetivo

Aprender a manejar estructuras de datos en Python y NumPy para análisis de datos e inteligencia artificial.
